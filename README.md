# Evenia Tickets - Web Dashboard

## Overview
A modern React-based web application for the Evenia Tickets platform. This frontend serves as the primary management portal for creators, organizers, and administrators to manage events and track sales.

## Key Features
- **Event Creation & Editing**: Multi-step forms for launching new events with multi-image galleries.
- **Admin Dashboard**: Comprehensive overview of stats, user management, and event approval.
- **Responsive Design**: optimized for desktop and tablet usage with a premium "Glassmorphism" aesthetic.
- **Real-time Updates**: Integration with the Laravel API for seamless synchronization.

## Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS / Custom CSS
- **Icons**: Lucide React

## Installation

1. **Clone the repository**:
   ```bash
   git clone <frontend-repo-url>
   cd frontend-evenia
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env` file based on `.env.example` and set your `VITE_API_BASE_URL`.

4. **Run Development Server**:
   ```bash
   npm run dev
   ```

## Workflow
- **Organizers**: Can create events, manage ticket types, and view sales history.
- **Admins**: Can validate events and manage the overall platform settings.
